addScore функция добавления очков игроку

Эта функция ищет игрока с заданным именем в таблице лидеров. Если игрок найден, его счет обновляется на указанное количество очков. Если игрок не найден, он добавляется в таблицу лидеров с указанным именем и счетом.

Аргументы
entriesуказатель на указатель на массив структур LeaderboardEntry, представляющий таблицу лидеров
sizeуказатель на переменную, содержащую текущее количество записей в таблице лидеров
nameимя игрока, для которого добавляются очки
scoreколичество очков, которое нужно добавить игроку
Возвращает
возвращает 0 в случае успешного добавления или обновления счета, -1 в случае ошибки

LoadLeaderBoard функция загрузки таблицы лидеров из файла

Эта функция открывает файл "leaderboard.txt", загружает данные о лидерах и их счетах из этого файла в массив структур LeaderboardEntry.

Аргументы
entriesуказатель на указатель на массив структур LeaderboardEntry, в который будут загружены данные о лидерах
sizeуказатель на переменную, в которую будет записано количество загруженных записей
Возвращает
возвращает 0 в случае успешной загрузки, -1 в случае ошибки
